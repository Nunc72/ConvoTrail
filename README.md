# ConvoMail

Conversation-centric email client — interactive UI mockup.

## Live

- **Mobile app**: [`/mockup-v4.html`](mockup-v4.html) — open on your phone and "Add to Home Screen" to install as a PWA
- **Phone-frame preview**: [`/mockup-v4-phone.html`](mockup-v4-phone.html) — best viewed on desktop

Visiting the root of the site auto-routes you to the right one.

## Stack

- React 18 (via CDN, no build step)
- Tailwind CSS (via CDN)
- Babel Standalone (JSX in the browser)
- PWA manifest + iOS Add-to-Home-Screen metadata

Everything is a single static folder — works on any static host.
